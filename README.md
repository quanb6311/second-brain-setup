Once the download finishes, you must prepare the file for use. Follow these instructions to ensure the workspace functions inside your folders.

1. Open your Downloads folder.
2. Right-click the file you just downloaded.
3. Select "Extract All" if it is a zipped folder.

5. Create a folder named "ai-memory" inside this location.

## 🚀 Running the application

The program works as a single-file system. It watches your notes for changes and feeds that information to your AI agents.

1. Double-click the application icon to start the process.
2. A black window will appear. This window shows the status of your connection.
3. Do not close this window while you work. It keeps your memory system active in the background.
4. If you need to stop the system, simply click the "X" in the top corner of that black window.

Use standard text editors to create your notes. You do not need special formatting, but using clear headings helps the AI find data faster. Think of this folder as your brain’s digital filing cabinet. Keep it tidy, and the AI will provide better answers.

## ❓ Frequently asked questions

**Does the software upload my private data?**
No. The system runs locally on your computer. Your notes stay within your control at all times.

**What happens if I lose my internet connection?**
The AI agents will not be able to read your memory folder while offline. However, your local notes remain perfectly safe on your hard drive.

**Can I move the folder later?**
Yes. You can move the folder to a different location, but you must restart the application so it knows the new file path.

**Does this work with my existing notes?**
Yes. You can drag your current files into the "ai-memory" folder to include them in your base of knowledge.

**Why does a black box appear?**
The black box is the terminal. It acts as the engine for the background processes. It logs activity to notify you if the system detects an error in your note files.
